OSQLinux
Biblioteca de clases en Java para la conectividad a base de datos alojadas en servidores con SQL.

Lanzamiento inicial: Jueves 27 de Marzo de 2025.
Edición: 1.3.2025 * serie SQLinux.
Proyecto: OSQLinuxLib.
Hospedaje: https://github.com/torrentelinux/OSQLinux
Descarga: Debe descargar los archivos CBaseDeDatos-abril2025.tar.xz, OSQLinuxLib-abril2025.tar.xz y LIBME-abril2025.tar.xz
Instala: Se sugiere que instale en Linux, manualmente, en el directorio /base/Proyecto/NetBeansLX/ y en Windows, en el directorio c:\base\Proyecto\NetBeans\
Sistemas: Se admiten los sistemas operativos Linux (preferentemente) y Windows Server/Desktop.
IDE: Apache NetBeans para Linux.
JDK admitidos:
	· OpenJDK Runtime Environment 21.3 (build 16.0.1+9).
	· Java(TM) SE Runtime Environment (build 17.0.12+8-LTS-286)
        · OpenJDK Runtime Environment (build 17.0.14+7-Ubuntu-124.04).
	· OpenJDK Runtime Environment Zulu23.28+85-CA (build 23+37).
        · OpenJDK Runtime Environment 23.0.2+7

Servidores SQL probados:
	· Bitnami LappStack ver. 7.4.25-0, 64 bits.
	· XAMPP for Linux 8.2.12-0, 64 bits.
	· Laragon Full 6.0, 64 bits.

JDBC admitidos:
	· PostgreSQL
	· MySQL
